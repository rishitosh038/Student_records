
## Project Overview
This project demonstrates how to manage structured student data using Python dictionaries and JSON file handling.  
It is designed for internship tasks, interview preparation, and beginner-level Python practice.

---

## Tools & Technologies
- Python 3.x
- VS Code / Any Python IDE
- JSON module (built-in)

---

## Project Structure
student_records.py  
students.json  
README.md  

---

## Project Objectives
- Store student details using a Python dictionary
- Perform CRUD operations (Create, Read, Update, Delete)
- Convert dictionary data into JSON format
- Save JSON data to a file
- Read JSON data back into Python
- Display clean and formatted output

---

## Features Implemented
1. Creation of student records using dictionary  
2. Accessing dictionary keys and values  
3. Updating student details  
4. Deleting student records  
5. Looping through dictionary items  
6. Converting dictionary to JSON  
7. Saving JSON data to a file  
8. Reading JSON data from file  
9. Printing formatted student information  

---

## How to Run the Project
1. Ensure Python is installed:
   python --version

2. Open the project folder in VS Code

3. Run the script:
   python student_records.py
